\section{Randomized Toy Experiments}

The previous section identifies Property X as the availability of close
substitute solutions in a decomposable feasible set. We now test this mechanism
in randomized toy families rather than relying on a single hand-designed
example. The goal is not to prove a theorem about every packing or path
problem, but to separate two effects: how often prediction noise changes the
selected solution identity, and how much true-objective regret that identity
change creates.

\paragraph{Randomized families.}
We compare three stochastic problem families. The first is a decomposable
packing family with 12 independent blocks and 4 candidate components per block;
a feasible solution chooses one component from each block. The second is a
stable-set realization of the same close-substitute mechanism: the conflict
graph is a disjoint union of 12 cliques with 4 candidate vertices per clique, so
an independent set can choose at most one vertex from each clique. With positive
weights, the oracle and predicted solutions choose one non-conflicting vertex
per clique. The third is a path-like control with 4 parallel $s$--$t$ paths,
each containing 12 edges; selecting a different path changes the entire coupled
route.

All families are controlled by two parameters. The parameter $\tau$ controls
the separation between alternatives, and $\sigma$ controls independent
prediction noise. For each $(\tau,\sigma)$ pair, we generate 500 random
instances. For each instance, we compute the oracle solution under true
coefficients, the two-stage solution under noisy predicted coefficients, the
normalized regret of the two-stage solution, whether the solution identity
changes, and the true oracle--second-best gap. We use the two-stage normalized
regret as the instance-level improvement ceiling: no decision-focused method
can improve over two-stage by more than the remaining gap to oracle.

\paragraph{Close substitutes: frequent identity changes, small regret.}
Table~\ref{tab:randomized-toy-summary} reports representative settings from the
randomized sweep. In the close-substitute regime ($\tau=0.02$,
$\sigma=0.10$), both decomposable packing and clustered stable set have
$100\%$ identity mismatch, but their mean normalized regrets are only $1.07\%$
and $1.05\%$, respectively, and all instances are within $5\%$ of oracle. Thus,
in these decomposable families, noisy two-stage predictions often change the
selected solution, but the selected solution is still a close substitute under
the true objective.

The shortest-path control shows why the problem label alone is not the full
explanation. In the close-substitute path regime, the path family also has
small mean regret ($0.47\%$), because the alternative paths are themselves
close. This supports the conditional version of our hypothesis: close
substitutes, not packing terminology by itself, limit the room for
decision-focused learning.

\paragraph{Separated alternatives: regret can reappear.}
When alternatives are more separated and prediction noise is larger
($\tau=0.30$, $\sigma=0.30$), the decomposable families no longer have uniformly
small regret. Mean regret rises to $7.71\%$ for packing and $7.57\%$ for stable
set, and the within-$5\%$ rates fall to $21.2\%$ and $24.8\%$. This does not
contradict Property X; it clarifies it. Decomposability makes local replacements
possible, but if prediction noise changes many local components at once, the
losses can accumulate and leave more room for decision-focused learning.

For the path family in the same separated regime, the unconditional mean regret
is $2.30\%$ because identity mismatch occurs in only $39.8\%$ of instances.
Conditional on a mismatch, however, the mean regret is $5.78\%$, almost the
same as the mean oracle--second-best gap of $5.80\%$. This is the path-like
contrast: when a ranking error redirects the route to a materially worse path,
the regret is controlled by the path-level best-second gap.

\paragraph{Takeaway.}
The randomized toy experiments support the conditional Property X hypothesis.
For combinatorial problems with close substitute solutions, two-stage
prediction can select a different solution identity while leaving little
decision regret for DFL to remove. This pattern appears not only in an abstract
decomposable packing model but also in a stable-set formulation with conflict
constraints. Conversely, when the selected alternative is materially worse, as
can happen when many local components are changed or when a path is redirected
to a worse route, the regret and the potential value of decision-focused
learning both increase.
